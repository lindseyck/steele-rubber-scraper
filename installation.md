# Installation Guide

## Prerequisites
- Python 3.x
- pip

## Installation Steps
1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/steele-rubber-scraper.git
   ```
2. Navigate to the project directory:
   ```sh
   cd steele-rubber-scraper
   ```
3. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
```
